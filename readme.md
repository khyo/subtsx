# SubTSX

A TSX gui platform based on [SubElectron](https://github.com/khyo/subelectron).


## Example Installation

```bash
python3 -mvenv .venv
source .venv/bin/activate
pip install https://github.com/khyo/subtsx
# verify cmdline tool is installed
subtsx --help
```
